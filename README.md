## ESlint

using flat config.

### vscode settings

- Enable `Eslint: Use Flat Config` on this workspace


## code signing

- https://www.electronjs.org/docs/latest/tutorial/code-signing

requirements

-  Apple Developer Program
   - Account Holder role is required
-  Set following environment variables
   - APPLE_ID
   - APPLE_ID_PASSWORD
   - APPLE_TEAM_ID
-  Xcode
